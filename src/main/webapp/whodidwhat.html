Our Group Contract

  1. Perform all work in a feature branch.
  2. Branch out from Developer.
  3. Never push into Developer or Master branch. Make a Pull Request.
  4. Delete local and remote feature branches after merging.
  5. Before making a Pull Request, make sure your feature branch builds successfully and passes all tests (including code style checks).
